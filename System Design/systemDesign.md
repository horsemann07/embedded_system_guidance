- Describe the basic architecture of an instrument cluster.

### Intermediate
- Design a complete digital instrument cluster system.
- Design an automotive gateway ECU connecting CAN, LIN, Ethernet, and diagnostics.
- Explain the OTA firmware update challenges in automotive systems.

### Advanced

- Design a vehicle OTA firmware update platform for millions of cars.

- Design a connected car telemetry and remote diagnostics platform.
- Design a zonal ECU architecture for next-generation EVs.
- Design a secure vehicle diagnostics system using UDS.
- Design a real-time CAN/FlexRay/Ethernet logging system.

### Expert

- Design a fail-operational braking system for autonomous vehicles.
- Design an automotive cybersecurity intrusion detection platform.
- Design a centralized vehicle compute platform replacing distributed ECUs.
- Design a high-reliability airbag controller system.
- Design a smart charging infrastructure communication platform for EVs.
- Design a real-time vehicle health monitoring platform.
- Design an automotive infotainment system using embedded Linux.
- Design a predictive maintenance platform for commercial vehicles.
- Design a V2X (Vehicle-to-Everything) communication system.

## Tips for Answering

- **For Basics**: Focus on understanding fundamental concepts, architecture patterns, and common challenges
- **For Intermediate**: Demonstrate system design thinking, trade-off analysis, and problem-solving approach
- **For Advanced**: Showcase scalability, reliability, and production-grade considerations
- **For Expert**: Discuss novel solutions, emerging technologies, industry standards, and visionary system design
